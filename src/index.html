STS2 Dashboard — Help

A quick reference for all features in this app.

Getting Started

History Folder On first launch, point the app at the folder where STS2 saves your .run files. Typical path on Windows: %APPDATA%\Roaming\STS2\history. The folder is watched live — new runs appear automatically.
Update Resources Downloads relic, card, enchantment, and event data (images + descriptions) from the STS2 wiki. No external tools required. Run this once before first use and again whenever new content is added to the game.
Change Folder Switch to a different history folder at any time via the header button.

Run List

Columns Character, outcome (Win / Loss / Abandoned), ascension level, floor reached, run duration, and date. Coop runs show all players and a CO-OP badge.
Sorting Click any column header to sort by that field. Click again to toggle ascending / descending.
Favorites Click the ★ star on a row to favorite a run. Favorited runs show a gold star. Use the Favorites only filter to see just your starred runs.
Copy Run Click 📋 on any row to copy the actual .run file to your clipboard — paste it directly into a folder, Discord, etc.
Open Run The 📂 Open Run header button lets you open any .run file from disk — including ones shared by other players — and view its full detail page. The run is loaded temporarily and does not get added to your run list or statistics; close the detail view and it's gone.
Pagination Runs are shown 12 per page. Use the page controls below the table to navigate.

Filters

Search Type a character name, relic name, or card name to filter runs that include it. Multiple tokens narrow the results further.
Mode Toggle between All, Solo, and Co-op to filter by game mode.
Character Filter to a specific character or keep All.
Ascension Click Ascension to open a multi-select of all ascension levels present in your runs.
Win Only / Abandoned Tick Wins only to show only victorious runs, or Include abandoned to show runs that were quit mid-way.
Min Duration Hide very short runs (e.g., crashes or accidental starts) by setting a minimum run length in minutes.
Favorites Only Show only runs you have starred.
Reset Filters Clears all active filters back to their defaults in one click.

Run Detail View

Opening a Run Click anywhere on a run row to open its detail page.
Journey Graph Shows your HP (green line) and max HP (blue dashed) across every node of the run. Click any node icon to see what happened at that stop — fight, event, shop, rest site, etc.
Relics All relics held at run end. Click any relic icon to see its description, rarity, and character.
Deck Full deck at run end. Upgraded cards are shown with their upgraded art. Click the enchantment badge on a card to view its description. Use the Sort button to order by type → rarity → alphabetical; click again to restore original order.
Co-op Player Tabs In co-op runs, tabs appear above the relics section to switch between players. Each tab shows that player's HP graph, relics, and deck.
Favorite & Copy Star a run or copy its .run file directly from the detail nav bar.

Insights Tab

Scope Insights always reflect whatever filters are currently active in the run list — so you can drill into stats for a specific character, ascension level, or game mode.
Relic / Card Win Rate Win rate for runs that included each relic or card. Use the Min samples slider to hide entries with too few data points.
Best Relic Pairs Win rate for runs that contained both relics simultaneously. Slider filters by sample count.
Best Card Duos & Trios Win rate for runs containing both (or all three) cards at once, computed over the top 30 / 20 most-seen cards to keep combinations meaningful.
Card Upgrade Value Compares win rate when a card was present in its base form vs. upgraded form, showing the delta. A large positive delta means upgrading that card tends to correlate with better runs.
Strike / Defend Removal Win rate split by whether you removed all copies of Strike or Defend from your deck vs. keeping at least one.
Deck Size vs Win Rate Win rate bucketed by final deck size, showing whether smaller or larger decks perform better in your runs.
Most Picked Relics & Cards Frequency ranking — how often each relic or card appeared across all filtered runs. Starter relics and starter cards are excluded.
Clickable Items Click any relic icon or name to see its full description. Click any card name to view the base and upgraded card images side by side.

A Pastebin developer API key is required to export pastes. To get one:

  1. Create a free account at pastebin.com
  2. Go to pastebin.com/doc_api#1 — your unique API dev key is shown at the top of that page
  3. Copy and paste it below
